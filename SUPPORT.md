# 支持 CCC

CCC 是一个开源的求职澄清与行动辅导项目。你可以免费阅读、复制、修改和使用公开文档、prompt、skill 与示例。

如果这个项目帮你理清了一点求职状态，欢迎用这些方式支持：

- 给项目 star；
- 提交 issue 或反馈，说明哪里有用、哪里太长、哪里不清楚；
- 分享给正在经历 Gap、转行、在职疲惫或求职迷茫的人；
- 在公众号、小红书或项目页通过赞赏支持维护。

赞赏是自愿的，不影响使用。CCC 不承诺面试、offer、薪资或求职结果，也不提供医疗、法律、签证或心理治疗建议。

## 反馈优先级

特别欢迎这些反馈：

- 哪一步让你更清楚；
- 哪一步太像模板；
- 哪些输出你真的会继续用；
- 哪些输出反而增加了焦虑；
- 是否误解了你的行业、岗位或缩写；
- 微信公众号 / Coze / 普通大模型使用时哪里卡住。

提交反馈前请先脱敏，不要公开真实简历、电话、邮箱、offer、合同、薪资截图、完整面试记录或公司内部信息。
