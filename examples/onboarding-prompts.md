# Onboarding Prompts

这些示例用于测试或演示 CCC。CCC 是 Career Cognition Compass 的缩写。这些示例故意保留一点混乱感，因为这个 Skill 的重点就是从“不知道怎么说”开始整理。

## 混乱开局

```text
Use $career-cognition-compass.

我不知道自己应该找什么工作。之前有一段 Gap，做过一些运营相关的事情，也自学过一点 AI 工具，但刷小红书和 LinkedIn 越看越焦虑。请先不要写简历，先帮我整理我已经有什么信息、还缺什么，以及下一步该做什么。
```

## 长期 Gap

```text
Use $career-cognition-compass.

我 Gap 超过一年了，现在想重新找工作，但不知道怎么解释，也不知道还能投什么岗位。请先帮我判断当前阶段、可迁移能力和最小行动，不要一开始就让我准备很多材料。
```

## 情绪波动与低能量启动

```text
Use $career-cognition-compass.

我已经很久没有稳定工作了，最近状态波动很大。每天都觉得应该找工作，但又不知道从哪一步开始，越刷社交媒体越焦虑。请先不要让我写简历，也不要给大计划，只帮我判断今天能做的 1 个很小的动作。
```

## 兼职/过渡工作

```text
Use $career-cognition-compass.

我可能需要先找兼职或临时工作过渡一下，但我担心兼职会消耗太多精力，影响正式求职。请帮我判断我适合找什么类型的兼职、哪些兼职应该排除，以及本周可以做的 1-3 个动作。
```

## 转行/转岗

```text
Use $career-cognition-compass.

我之前所在行业下行，想转行或转岗，但不知道自己的技能能迁移到哪里。请帮我把经历拆成证据和能力，给出 1-3 个可以先验证的方向，并告诉我还需要补什么证据。
```

## 职业族群筛选

```text
Use $career-cognition-compass.

我现在完全不知道该看什么岗位。运营、销售/BD、产品/项目、数据、市场、设计、职能支持、技术入门、AI 应用这些方向我都分不清。请先不要给我一长串岗位，帮我根据已有经历筛出最多 3 个角色族群，并说明每个方向的证据、缺口和 7 天验证动作。
```

## 快速就业

```text
Use $career-cognition-compass.

我现在更需要尽快找到工作。每天大概能花 2 小时求职。请帮我选择更实际的方向，做一个不超过 14 天的计划，只输出我真的会用到的材料。
```

## 校招新人

```text
Use $career-cognition-compass.

我是应届生，有课程项目、一段实习、两个比赛/获奖经历，但不知道怎么和岗位联系起来。请帮我整理求职证据，判断可以优先尝试哪些岗位。
```

## 在职换工作

```text
Use $career-cognition-compass.

我现在在职，想换工作，但不确定是换公司、换岗位，还是换行业。我也担心求职信息泄露。请先帮我澄清真正想改变的部分，并给一个低风险的准备顺序。
```

## 在职焦虑 / 工作疲惫

```text
Use $career-cognition-compass.

我现在还在职，但最近工作让我很焦虑、很累，下班后也恢复不过来。我还没想清楚自己是需要休息、调整边界、换工作，还是干脆离职。请先不要让我写简历，也不要直接劝我辞职，先帮我做在职焦虑/疲惫应对卡，拆出压力来源、今天能做的小动作，以及下一步是否需要进入离职/裸辞判断。
```

## 在职离职 / 裸辞犹豫

```text
Use $career-cognition-compass.

我现在在职，但已经很想离职，甚至想裸辞。可是我没有 offer，也不确定自己是需要休息、换公司、换岗位，还是整个方向都不对。请不要直接劝我辞或不辞，先帮我做离职/裸辞判断卡，评估现金流、现职损耗、市场验证、法律/合同风险和 7-14 天内可以做的验证动作。
```

## 缩写歧义

```text
Use $career-cognition-compass.

我说 SP 的时候指的是 SAS Programmer。我也在考虑 PM，但我不确定自己说的是产品经理还是项目经理。请先帮我建立一个小术语表，再比较这些方向需要的技能和证据。
```

## MBTI / 星座 / 性格标签

```text
Use $career-cognition-compass.

我是 INFP，也会参考星座来理解自己。我不想让这些标签直接决定职业方向，但我觉得它们能说明我对工作氛围、压力和协作方式的感受。请帮我把这些标签转成可验证的工作偏好、需要避开的环境和下一步要补充的真实例子。
```

## 目标公司准备

```text
Use $career-cognition-compass.

我想准备某一家公司的某个岗位。我会贴 JD。请不要直接生成套话，先帮我判断这个岗位重视什么、我有哪些匹配证据、最大的缺口是什么，以及面试前 1-3 个最该准备的问题。
```

## 使用环境分流

```text
Use $career-cognition-compass.

我想把这个求职流程放到不同 AI 工具里用：国内大模型、海外大模型、普通 LLM 对话、Coze Bot、还有 Codex 这种文件代理。请帮我严格区分每种环境适合做什么、不适合做什么、需要用户提供什么，以及隐私边界。
```

## 面试复盘

```text
Use $career-cognition-compass.

我刚面试完，只记得几个关键词：留存、SQL join、用户分层、dashboard，不记得完整问题。请帮我还原可能的题型，整理进我的知识库，并准备 1-2 个高概率问题的回答思路。
```
