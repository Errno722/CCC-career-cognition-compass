# CCC Local Web App

这是 CCC 的本地静态 Web 应用版本。它不接入 API，不上传数据，主要用于把混乱输入整理成可复制给 LLM / Agent 的任务包。

## 使用方式

直接用浏览器打开：

```text
webapp/index.html
```

也可以在本地启动一个静态服务：

```bash
cd webapp
python3 -m http.server 5173
```

然后访问：

```text
http://localhost:5173
```

## 适合做什么

- 粘贴语音转文字、碎片经历、JD、面试回忆或简历片段；
- 本地判断当前任务更像方向澄清、职业定位、简历材料、JD 分析、面试复盘或行动计划；
- 检测是否可能需要职业定位 / 候选人叙事重建；
- 生成可复制给普通 LLM、Coze、Codex 或 Claude Code 的任务包；
- 在浏览器本地保存简短知识库草稿。

## 边界

- 它不是聊天机器人，不会直接调用大模型；
- 它不会自动投递、自动私信 HR 或承诺面试结果；
- 它不替使用者做最终决定；
- 粘贴内容前仍然需要脱敏。
