# Coze 商店发布文案

## 名称

```text
CCC 求职澄清智能体
```

## 副标题

```text
从混乱求职状态到清楚的下一步
```

## 简短介绍

```text
CCC 是 Career Cognition Compass。它不是一上来写简历的工具，而是先帮你整理求职状态、经历证据、可迁移能力、JD 重点、面试复盘和下一步行动。
```

## 详细介绍

```text
CCC 适合正在经历 Gap、转行转岗、校招迷茫、在职疲惫、离职犹豫、JD 分析、面试复盘或不知道该投什么岗位的人。

你不需要先准备完整简历，也不需要填表。可以直接发一段很乱的话，CCC 会先帮你整理：

1. 我听到的重点
2. 当前最可能卡住的问题
3. 还缺的关键信息
4. 下一步 1-3 个小动作

CCC 会提醒你保护隐私，不建议发送身份证、电话、邮箱、offer、合同、薪资截图、完整简历或公司内部信息。

如果你已经有多份简历，CCC 会区分中性主简历和某个 JD 的定制版本，避免上一份岗位偏向影响下一份简历。

CCC 会注意你的状态，但主要功能不是情绪陪伴或心理咨询，而是帮你把求职信息整理成事实、选择和下一步行动。

如果你一开始就想改简历，CCC 会先回应简历修改需求，要求脱敏后的简历片段和目标岗位，再给可编辑的修改建议。

CCC 只提供整理、分析和建议，不承诺面试、offer、薪资或求职结果，也不替你做离职、裸辞、offer 选择等最终决定。
```
