# CCC Coze 公开说明

CCC 是 Career Cognition Compass 的缩写。这个页面说明如何在 Coze 商店或公众号入口使用 CCC。

CCC 已经发布到公众号，也可用于 Coze 商店展示。这个公开仓库只保留使用方法、适合场景和隐私边界，不展示智能体搭建、部署、公众号后台配置或服务器配置过程。

CCC 会注意使用者当前状态，但它不是情绪陪伴或心理咨询工具。它主要帮助使用者把混乱信息整理成事实、选择、风险和下一步行动。

如果使用者一开始就明确说要改简历，CCC 会先进入简历修改流程，不会强行要求先做职业澄清。

## 适合谁

CCC 适合这些使用场景：

- 刚开始找工作，脑子里很乱，不知道从哪里说起；
- 长期 Gap 后想重新进入求职节奏；
- 转行、转岗，或原行业下行，不知道能力能迁移到哪里；
- 校招新人，不知道如何整理课程、实习、项目和获奖经历；
- 在职很累，不确定该换工作、换方向、休息，还是先观察；
- 已经有目标岗位、目标公司或 JD，需要准备面试和投递；
- 面试后只记得关键词，想复盘但不知道怎么整理；
- 想优化简历表达，但不想编造经历；
- 已经有多份按不同 JD 改过的简历，需要避免上一版偏向影响下一版；
- 需要兼职、临时工作或过渡工作来稳定现金流和节奏。

## 在 Coze 商店怎么用

如果你是从 Coze 商店进入 CCC，可以直接开始对话，不需要先准备完整简历或填写表格。

Coze 对话页比公众号更适合承接较长输入。你可以粘贴一段混乱文字、录音转文字、长 JD、面试回忆或多段背景信息。内容不需要写成正式材料，但发送前要先脱敏。

适合先发这些内容：

```text
我现在很乱，gap 一年多，不知道还能投什么。
```

```text
我想改简历。我会提供脱敏后的简历片段和目标岗位。
```

```text
这是一个产品经理 JD，帮我看硬技能、工具要求和面试可能问什么。
```

## 在公众号怎么用

如果你是从公众号入口进入 CCC，可以直接发送文字。公众号更适合很小的一句话、几个词或一段短材料，适合在手机上低门槛开始。

目前订阅号功能受限，没有自定义菜单入口。使用时不需要找菜单，直接在公众号聊天框发送关键词、短句或脱敏材料即可。公众号文章、简介或自动回复可以作为入口说明。

比如：

```text
我现在很乱
```

```text
gap 一年，不知道还能投什么
```

```text
看到一个 JD，不知道怎么准备
```

公众号不建议一次发送特别长的录音转文字。长文本更适合放到普通 LLM、Coze 对话页、Codex 或 Claude Code 这类 agent 环境里处理。

如果公众号里出现“思考中，请回复继续”或类似提示，这通常是平台的回复时间限制，不代表内容出错。直接回复“继续”即可接着上一段；为了减少这种情况，建议把长 JD、长简历或长语音转录分成几段发送，或按 CCC 给出的 1 / 2 / 3 选项继续。

推荐顺序：

```text
1. 先发当前状态或问题
2. 再发脱敏后的 JD / 简历片段 / 面试关键词
3. 如果回复让你选择“JD / 简历 / 面试 / 方向”，按你当前最需要的方向回复
```

公众号场景下，复杂任务会被拆成较短回复。例如长 JD 不会一次性完整分析，而是先拆核心任务、硬技能和最该补的缺口。

## 怎么开始整理

使用者不需要先整理简历，也不需要填写表格。可以直接发一段混乱文字、语音转录、碎片想法，或者不带标点的一串词。

如果更习惯说话，可以先录音。打开手机里的录音工具，对着手机说 5-10 分钟，把现在的状态、之前做过的事情、为什么想找工作或换方向、看过哪些岗位、最卡住的地方、每天大概能用多少时间都说出来。

录音前先注意脱敏。不要说出电话、邮箱、身份证、薪资、offer、合同、客户信息、公司内部信息或任何不希望被平台看到的内容。

录完后，用手机自带语音转文字、输入法语音转写、会议记录工具或其他转录工具生成文字。转录内容可以有重复、停顿和口语，不需要先改成正式表达。快速看一遍，把敏感信息替换成 `[城市]`、`[公司A]`、`[岗位B]`、`[项目代称]` 或 `[数据已脱敏]`，再发送给 LLM 或 Agent。

这个录音过程本身也有用。很多人在说出来的过程中，已经开始发现自己真正卡住的地方。

如果暂时说不清楚，也可以一句一句发。CCC 会先把碎片整理起来。

示例：

```text
我现在很乱，gap 一年多，之前做过运营，也学过一点 AI，不知道还能投什么。
每天大概能花 2 小时找工作，但越看社交媒体越焦虑。
我想先把状态、经历和下一步整理清楚。
```

也可以更短：

```text
gap 一年 运营 ai 转行 不知道投什么
```

或者一句一句开始：

```text
gap 一年
之前做运营
学过一点 AI
不知道还能投什么
```

CCC 会先整理：

```text
1. 我听到的重点
2. 还缺的关键信息
3. 初版经历证据
4. 初版硬技能知识库
5. 初版术语/缩写表
6. 下一步最该补充的 1-3 件事
```

## 使用前提醒

CCC 只提供整理、分析和建议，最终决定仍由使用者根据现实情况自己做。

请不要在对话中提交：

- 身份证、护照、签证号码；
- 电话、邮箱、完整住址；
- offer、合同、薪资截图；
- 完整面试逐字稿；
- 公司内部信息、客户信息或未公开项目；
- 任何不希望被平台或他人看到的隐私内容。

可以用这些方式脱敏：

```text
[城市]、[公司A]、[岗位B]、[项目代称]、[具体数据已脱敏]
```

## Coze / 公众号入口会做什么

公开入口适合做轻量整理：

- 混乱输入整理；
- 方向初筛；
- 可迁移能力判断；
- JD / 目标公司准备；
- 面试等待与复盘；
- 简历表达边界；
- JD 简历修改补丁；
- 简历能力聚焦检查，例如模块太多、个人总结太泛、与 JD 不够贴合时，先找 2-3 个最相关能力；
- 简历版本隔离检查；
- 在职焦虑、离职犹豫、兼职过渡的初步整理。

## 它不会做什么

CCC 不做这些事：

- 不自动投递职位；
- 不自动私信 HR；
- 不承诺面试、offer、薪资、签证或平台曝光；
- 不编造学历、公司、岗位、项目、奖项、证书、数据、客户或技术能力；
- 不替使用者做离职、裸辞、接受 offer、拒绝 offer、薪资谈判等最终决定；
- 不做心理诊断、治疗、危机干预、法律建议、医疗建议或签证结论。

## 开源使用与赞赏

CCC 采用开源 + 支付宝自愿赞赏的方式维护。你可以免费使用公开文档、prompt、skill 和 demo。

如果这个项目对你有帮助，可以通过这些方式支持：

- star / fork 项目；
- 提交反馈或改进建议；
- 分享给需要的人；
- 通过支付宝自愿赞赏。

赞赏不等于结果承诺。CCC 仍然不提供虚构经历、代投递、代沟通 HR、面试保证、offer 保证或薪资保证。
